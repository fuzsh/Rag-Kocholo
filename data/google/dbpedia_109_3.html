What is the character representation of ASCII?
The standard ASCII character set is only 7 bits, and characters are represented as 8-bit bytes with the most significant bit set to 0 . Modern computers almost universally use 8-bit bytes, and the extended ASCII character set includes 127 more 8-bit characters, where the most significant bit is set to 1.

What is the structure of Unicode?
The Unicode codespace is divided into 17 planes, numbered 0 to 16 . Plane 0 is the Basic Multilingual Plane (BMP), and contains the most commonly used characters. All code points in the BMP are accessed as a single code unit in UTF-16 encoding and can be encoded in one, two or three bytes in UTF-8.

What are the formats of Unicode encoding?
