What is the stratum hierarchy of NTP?
NTP uses a hierarchical, semi-layered system of time sources. Each level of this hierarchy is termed a stratum and is assigned a number starting with zero for the reference clock at the top . A server synchronized to a stratum n server runs at stratum n + 1.

What is stratum 0 in NTP?
Network Time Protocol (NTP) uses "Stratum" to describe the distance between a network device and an authoritative time source. Stratum 0 is the GPS Satellite itself . Stratum 1 NTP Time Servers reference GPS Satellites directly and are considered the most accurate, because there is no network delay.

What are the stratum layers of NTP?
NTP uses stratums 1 to 16 to define clock accuracy, as shown in Figure 32. A lower stratum value represents higher accuracy. Clocks at stratums 1 through 15 are in synchronized state, and clocks at stratum 16 are not synchronized.

What is the difference between NTP and SNTP?
NTP can use multiple time servers to keep the system synchronised, but SNTP is based on a single time server . Extra servers can be used as back-ups, but SNTP can't communicate with multiple servers to work out which one is the most accurate. NTP has extra security features, like secure client-server authentication.
