What is a .GIF file?
GIF stands for Graphics Interchange Format. GIF is a raster file format designed for relatively basic images that appear mainly on the internet . Each file can support up to 8 bits per pixel and can contain 256 indexed colors. GIF files also allow images or frames to be combined, creating basic animations.

Biography. Stephen Earl Wilhite was born in West Chester Township, Ohio, on March 3, 1948, the son of Anna Lou (Dorsey), a nurse, and Clarence Earl Wilhite, a factory worker. Wilhite's team at CompuServe developed the GIF (Graphic Interchange Format) in 1987.

What is the header of a GIF file?
GIF files start with a fixed-length header ( "GIF87a" or "GIF89a" ) giving the version, followed by a fixed-length Logical Screen Descriptor giving the pixel dimensions and other characteristics of the logical screen.
