The surname Durning is of English origin. It is derived from the Old English personal name "Dyring", which itself is derived from the Old Norse name "Dyri", meaning "deer". The name would have originally been used to describe someone who was keen on deer hunting or had some association with deer. Over time, the name evolved into the surname Durning.
