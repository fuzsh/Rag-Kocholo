Province Valencia is an administrative unit in Autonomous community Valencian Community.

To make a local call (from Spain to other place in Spain) you need to dial the area dial code and then the subscriber number, even the subcriber is located in same place from where you are dialing.

To call Spain from another country (international call) - dial 00 (or other prefix if it is different), folowed by the country dial code of Spain (34), then code of dialed location and the subscriber number.

To make a local phone call from Spain to number in Province Valencia: 960-963xxx

To make an international phone call from a foreign country to number in Province Valencia: 0034960-963xxx

The phone code of municipalities in Province Valencia - 960-963.
