Open Source Software

Satlab has released and contributes to a number of open source projects. Please see our GitHub page for a full list of public repositories.

CSP

Satlab is co-maintainers of the Cubesat Space Protocol (CSP), a lightweight network-layer delivery protocol designed for CubeSats. All Satlab products are operated using CSP via CAN, RS-422 or Ethernet.

Upstream repository: http://www.libcsp.orgSatlab development repository: https://github.com/satlab/libcsp

Slash

Slash is a command line editor and command parser library for embedded systems. It is especially suited for serial debugging interfaces on memory constrained systems.
