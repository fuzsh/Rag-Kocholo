Data

The dataset consits of 7,672 Natural Language Landmark Navigation Instructions and corresponding route paths in OpenStreetMap. The data collection is described in the paper by (Schumann & Riezler, 2021), please cite it if you use the data.

Visualization of the routes in the dataset and a demo of the model trained on it can be found here.
