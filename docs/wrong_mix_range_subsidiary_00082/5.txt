If you search for a document title (with or without quotes), you will see documents that match that title first and then other results.

If you search for that title in double quotes it will only show results with that exact phrase with title matches first. You can add additional queries like this

"exact phrase" keyword -negativeword

This will look for that the exact phrase that also contains keyword, but does not contain the negativeword

You can also search for multiple exact matches. For example if you were looking for security deposit language in commercial leases you could search for:

"security deposit" "commercial lease"

Use the Advanced Search Box to drill down to specific documents based on agreement type, governing law, law firms, parties, SIC code and date.
