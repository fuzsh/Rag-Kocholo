JTS Topology Suite (JTS) formerly known as Java Topology Suite is an open source Java software library that provides an object model for geometries and functions to work with them. It is a project of LocationTech.

JTS has the following features:

conforms to the Simple Features Specification for SQL published by the Open GIS Consortium

provides a complete, consistent, robust implementation of fundamental 2D spatial algorithms

fast enough for production use

written in 100% pure Java
