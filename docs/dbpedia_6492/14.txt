What do Norway SWIFT codes look like?

The SWIFT code consists of 8 to 11 characters (letters and digits) and for Norway the SWIFT code has the following format:

First 4 characters (only letters) - bank code. A shortened version of the bank´s name.

Next 2 characters (only letters) - Norway´s ISO code.

Next 2 characters (letters and digits) - location code.

Last 3 characters (letters and digits) – bank branch (optional).
