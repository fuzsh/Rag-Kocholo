What is the postal code of Gombe State? Understand postal codes and find out the postal code of Gombe State.

Postal codes in Nigeria are numeric, consisting of six digits. The Nigerian Postal Service (NIPOST), divides the country into six geo-political regions. The first digit of Gombe State postal code represents its region. The second and third digits of the postal code of Gombe State, combined with the first, are the dispatch district for outgoing sorting. The last three digits represent the delivery location.
