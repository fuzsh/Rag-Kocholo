The 2005 PC Magazine Utility Guide issue is out and Stardock's Object Desktp won the Editor's Choice Award. Stardock's popular suite of Windows enhancement utilities also won the award last year. This year's Object Desktop includes a major new version of DesktopX, a new program called RightClick another new program called IconX along with significant enhancements to core features such as WindowBlinds, SkinStudio, and Theme Manager.

For Object Desktop 2005, Stardock also recently added a "Desktop Gadgets" component which takes some of the popular DesktopX widgets and transforms them into stand-alone programs for users to use on their desktops even if they don't have DesktopX installed.
