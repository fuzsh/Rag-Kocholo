Yahoo! subsidiary Zimbra on Wednesday announced that its Zimbra Collaboration Suite (ZCS) now supports Safari 3, iCal 3is CalDAV format, and usage on the iPhone, via the Zimbra Mobile HTML client. ZCS uses an AJAX interface that includes email, calendaring, Zimbra documents, and customizable mash-ups called Zimlets.

ZCS 5.0 Open Source Edition is free, while the ZCS 5.0 Network Edition is available with a 60-day free trial, after which time users can contact Zimbra for a quote. Educational discounts are available.
