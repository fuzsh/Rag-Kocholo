General Commands Manual NAME

remuco-amarok - A daemon which allows control to Amarok via bluetooth SYNOPSIS

remuco-amarok DESCRIPTION

remuco-amarok is a daemon which listens for connections via Bluetooth from a Remuco client, which runs on a mobile device. This allows for remote control of Amarok from the said mobile device. COPYRIGHT
