The source codes for Clonezilla

Clonezilla source codes actually are the running programs themself, because they are written in scripts (bash or perl).

For partclone, ntfsclone, partimage, dd, Debian live or other programs, you can browse their own project websites about the source codes.

If you are looking for the tarball of Clonezilla package, check this.

If you are looking for the tarball that contains all the sources packages included in Clonezilla live, they are only available on the NCHC's repository.
