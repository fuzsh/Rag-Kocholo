Nucloud is a file-hosting service provided by Nusoft Foundation, and created on September 3, 2007, which enables user cloud storage, file sharing, and collaborative editing. Files shared publicly can be searched within Google, Bing, Yahoo!, DuckDuckGo, and Zuma Search.

Image files (JPEG, JPG, PNG, BMP, TIFF, SVG, ICO, and ICNS)

Storage[]

Nucloud offers 1,000 EB of storage for every Nusoft.net account.
