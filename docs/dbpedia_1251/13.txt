MediaCoder is a general-purpose batch media transcoding software. It supports a wide range of existing audio/video compression formats and container formats with various filtering, post-processing and stream manipulation features. A rich set of adjustable parameters are provided to cover every detail of the the transcoding process. MediaCoder is suitable for all kinds of transcoding applications, including transcoding for device playback and for Internet streaming, especially with batch processing demands.
